# Credentials (not committed)

Copy the example files and fill in your keys. These paths are gitignored.

```bash
cp apikey.txt.example apikey.txt
cp exa_key.txt.example exa_key.txt
```

Configure OpenCode / OpenRouter to use your OpenRouter API key (see [OpenCode docs](https://open-code.ai/)).

`apikey.txt` — OpenRouter or primary LLM provider key.  
`exa_key.txt` — Exa API key for web search MCP (optional).
